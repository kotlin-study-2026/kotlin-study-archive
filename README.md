# 🎯 kotlin-study-archive

Kotlin 스터디 주차별 발표 자료 및 코드 아카이브입니다.

---

## 📌 자료 등록 방법

### 1️⃣ 브랜치 생성
```
week01/이름
```
예시: `week01/윤지`

### 2️⃣ 파일 추가
해당 주차 폴더에 파일 추가
```
week01/
├── 이름_발표자료.pdf
└── 이름_week01.kt
```

### 3️⃣ 이슈 등록
- Issues 탭 → New issue
- 템플릿 선택 후 내용 작성
- PDF는 이슈에 첨부

### 4️⃣ PR 생성
- `week01/이름` → `main` 으로 PR
- 이슈 번호 연결 (`closes #이슈번호`)
- 리뷰 후 merge
